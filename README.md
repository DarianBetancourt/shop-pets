# PetShop
é um API para gerenciamento e controle de estoque, que permite cadastrar, listar,editar e apagar, produtos e serviços da PetShop 

- ```git clone https://github.com/DarianBetancourt/shop-pets.git```
- ```cd shop-pets/```
- ```npm install```
- ```npm run dev```
